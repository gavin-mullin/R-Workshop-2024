Chapter 3 Getting started

3.1 Script vs. notebook vs. markdown?

There are essentially three different file types to note when it comes to creating R code: script, notebook, and markdown. To put it simply, script is all manual and notebook/markdown are interactive scripts. The exact differece isn’t important right now, but some people have preferences for one of the other. This course will be using notebooks to easily send and access “assignments”.

3.2 Basic operations

Here, we will be going over the very basic operations involved in coding. Most of coding relates to mathematics and logic! For example, below is a table that features basic math (addition, subject, etc.), how it is written in R (pay attention to the symbols used), and its result.

Math R code Result
\(8 + 4\) 8 + 4 12
\(8 - 4\) 8 - 4 4
\(8 \cdot4\) 8 * 4 32
\(8 / 4\) 8 / 4 2
\(8^4\) 8 ^ 4 4096
